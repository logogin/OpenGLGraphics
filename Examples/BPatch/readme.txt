OpenGL BPatch (Bezier Patch)

Author : Mustata Bogdan (LoneRunner)
Email  : lonerunner@planetquake.com
Web    : http://www.cfxweb.net/lonerunner/
Date   : 29/10/00
=============

   This is a random surface generator which uses 4 bezier patches to make
a cool surface. The demo suport light, envmap and can load different
textures.
   Source code is included in the ZIP.

   When compiling make sure that you link to opengl32.lib and glut32.lib.

=====
Usage
=====
  
Esc		-- Exit
F		-- Full Screen
E		-- EnvMap On/Off
T		-- Textured On/Off
Space		-- Generate surf
Mouse Right	-- Zoom
Mouse Left	-- Rotate
Mouse Middle	-- Pop-up Menu
1,2,3		-- texture: phong.tga, 64firery-1.tga, testenv.tga
